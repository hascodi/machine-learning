Hoofdstuk 8 TensorFlow

Leerdoel 8.1 Ontwikkelt het vermogen om zelfstandig technisch-uitdagende online workshops uit te voeren (EA_LD755).

8.1 Inleiding

TensorFlow is een ML-gebaseerde bibliotheek voor dataflow.(nl: datastroom-georiënteerd programmeren) in 2015 vrijgegeven door het Google Brain team. Het laat toe om het principe van differentieerbaar programmeren (eng: differentiable programming) en symbolische wiskunde (eng: symbolic math) toe te passen. TensorFlow is met andere woorden een bibliotheek om tensoren door middel van een dataflow te verwerken. Een tensor kan je voorstellen als een schaalbare vector. In de Appendix vind je meer uitleg over wat een tensor is en wat het verschil precies is met een reguliere vector.

8.2 Basistaken in TensorFlow

Hieronder volgt de implementatie van een aantal veel voorkomende taken in TensorFlow:

Definiëren van een constante:

import tensorflow as tf

answer = tf.constant(42)

Definiëren van een variabele:

answer = tf.Variable(42, name = "The answer to life, the universe and everything")

Merk hier op dat de namespace van Python en TensorFlow volledig gescheiden zijn en dat biedt technische voordelen, maar het maakt het ontwikkelen een beetje stroef, want je moet voor elke variabele de TensorFlow naam toevoegen.

Definiëren van een eenheidsmatrix (eng: identity matrix) of nulmatrix (eng: zero matrix):

theta = tf.Variable(tf.ones(shape = (2,2)), name = "theta")
theta_0 = tf.Variable(tf.zeros(shape = (2)), name = "theta_0")

Definiëren van een functie, hier om een matrix te vermenigvuldigen met een co-vector (zie Appendix):

@tf.function
def forward(x):
  return theta * x + theta_0

De lijn met @tf.function is een decorator die aan TensorFlow de mogelijkheid geeft om de functie te optimaliseren. We kunnen nu de gedefinieerde functie eenvoudig uitvoeren:

forward([1,0])
## <tf.Tensor: shape=(2, 2), dtype=float32, numpy=
## array([[1., 0.],
##        [1., 0.]], dtype=float32)>

We zien dat het resultaat een instantie van het object (of de klasse) Tensor is, een klasse die gedefinieerd is binnen de tensorflow bibliotheek module[^1]. Willen we hieruit een array zelf extraheren, ku In plaats van nullen en enen kunnen we matrices ook vullen met normaal-verdeelde gegevens, bijvoorbeeld voor het initialiseren van het netwerk:

theta = tf.Variable(tf.random.normal([5, 5], stddev = 2, name = "theta"))
